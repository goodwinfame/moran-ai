## Logo（SVG 矢量版）

> **设计概念**：墨滴落纸晕染。中心深色墨滴落点向外扩散出同心菱形晕染层，逐层变淡。
> 几何感：圆角菱形（非圆形），保持棱角与抽象感。
> 配色：navy(#1A202C) → teal(#3D5A6E)，渐变 + 半透明叠加。

| 文件 | 用途 | 描述 |
|------|------|------|
| **logo-icon.svg** | 图标（favicon、小尺寸） | 墨滴晕染同心菱形，4层渐变扩散 |
| **logo-text.svg** | 文字组合（导航栏、启动页） | 图标 + 墨染(衬线) + MORAN(小字下方)，紧凑双层 |
